Criação de Tabelas no HTML.

Definição

Para criarmos tabelas no HTML temos varias tags que podem e devem ser utilizadas. Abaixo veja o exemplo de algumas delas e como utilizar.

  1. A principal tag utilizada para criar uma tabela é: <table></table>.
  2. Para criarmos as linhas de uma tabelas utilizamos a tag: <tr></tr>. Que significa Table Row ou linha da tabela.
  3. Agora para criarmos a coluna de uma tabelas utilizamos a tag: <td></td>.Table Data ou dados da tabela define uma célula dentro da tabela.

    4. Aleém das celulas de dados temos as celulas de cabeçalho, assim como temos cabeçalho de uma pagina HTML. Vejamos abaixo.

  4. Para definir o cabeça de uma tabela no HTML utilizamos a tag: <th></th>.
